# Control Structures

## blocks
- {} can be used to create isolated block scope

## loops (for)
- two types
    - `for i`
    - `for range`


## If
- types
    - `if`
    - `if else`
    - ``if ... else if`

## switch
- `switch typeX`

## labels
- break statement
  - switch, for loops
  - label optional
- continue statement
  - for loops
  - label optional
- goto statement
  - function scope, anonymous functions are excluded from scope
  - used to move execution to the labeled statement
  - label IS required
